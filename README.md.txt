# AI Powered Space Debris Risk Analyzer

This project uses Machine Learning (Decision Tree) to classify space debris risk into Low, Medium, and High categories.

## Features
- Synthetic dataset generation
- Decision Tree classification
- Model validation
- Accuracy evaluation

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Generate dataset:
   python dataset_generator.py

3. Train model:
   python train_model.py
